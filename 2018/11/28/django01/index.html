通过django在Mysql数据库中创建表
1.在应用下的models.py文件中编写数据模型
创建简单的学生表格模型
class Student(models.Model):
name = models.CharField(max_length=10,unique=True)
age = models.IntegerField(default=18)
gender = models.BooleanField(default=1)

2.在项目下的settings.py文件中的INSTALLED_APPS下最后一行添加’应用的名字’

在DATABASES 中更改数据库的信息 

DATABASES = {
  'default': {
      'ENGINE': 'django.db.backends.mysql',
      'NAME': "dj7",
      'HOST':"localhost",
      'USER':'root',
      'PASSWORD':"123456",
      'PORT':3306
  }
}

3.更改项目下init.py文件，在里面倒出pymysql
pymysql.install_as_MySQLdb()

4.python manage.py makemigrations 生成迁移文件
此时在应用下的grations下面就会生成一个0001_initial.py文件

5.python manage.py migrate 迁移文件
此时就会在dj7这个数据库下生成一张与模型对应的表格，自动生成主键，主键自动增长

二、字段的定义

字符串： CharField
整型 ： IntegerField
布尔 : BooleanField
日期 ： DateTimeField、DateField、TimeField
自增 ： AutoField
图片 : ImageField

约束定义
是否唯一 unique
是否为空 null
默认值 default
默认时间 auto_now_add
修改时间 auto_now

查询 –filter() ==>’select * from xxx’
一、
模型名.objects.filter(xxx=’筛选’)
filter方法：
查询结果可以不存在 或者可以为多个
如果查询结果不存在，返回一个空的列表

二、
模型名.objects.get(xxx=’筛选’
get 方法：
获取唯一的满足条件的对象，且查询的条件必须存在
如果查询结果不存在或者结果为多个都报错

查询结果的筛选：
first()：查询结果中的第一个
last():查询结果中的最后一个

拿到全部表格对象：
模型名.objects.all()

多条件筛选：
模型名.objects.filter(xxx=’筛选’).filter(xxx=’筛选’)
模型名.objects.filter(xxx=’筛选’，xxx=’筛选’)

exclud():查询不包含该条件的结果

# 查询不叫大明的学生信息
stus = Student.objects.exclude(name='军军')

values():选择查看结果对象的信息

# 通过values()查看学生对象的所有信息
stus = Student.objects.filter(name='军军').values()
# 通过values()查看学生name,id的信息
stus = Student.objects.filter(name='军军').values('name','id')

exists():

# 判断name = 军军的学生存不存在，存在返回True，不存在返回False
stu = Student.objects.filter(name='军军').exists()

排序：
排序 order_by
降序 –> -id ,在sql中id desc
升序 –> id , 在sql 中id asc

#查询所有的Student的对象，以升序排序
stus = Student.objects.all().order_by('id')
#查询结果以age的大小（逆序）排序
stus = Student.objects.all().order_by('-age')

基于like模糊查询：

contains:区分大小写;icontains:大小写不敏感
name__contains='大' -->相当于是like '%大%'
stus = Student.objects.filter(name__contains='大')

endswith:区分大小写；iendswith：不区分大小写
name__endswith='大'-->相当于是like ‘%大’
stus = Student.objects.filter(name__endswith='大')

startwith:区分大小写，istartswith：不区分大小写
name__startswith='大'-->相当于是like ‘大%’
#stus = Student.objects.filter(name__startswith='大')

in: 集合运算

#sql :select * from Student where id in (1,2,3);
# 查询id在[1,2,3] 中的对象
stus = Student.objects.filter(id__in=[1,2,3])
# 查询pk（主键）在[1,2,3]中的对象
stus = Student.objects.filter(pk__in=[1,2,3]

比较运算:

大于：__gt__    ；大于等于 ：__gte__
小于：__lt__    ；小于等于 ：__lte__
#查询age小于30的对象
stus = Student.objects.filter(age__lt=30)

聚合函数：

# 获取所有对象age的平均值
avg_age = Student.objects.all().aggregate(Avg('age'))
print(avg_age)
# 查询所有对象age的总和
sum_age = Student.objects.all().aggregate(Sum('age'))

Q对象（用于执行复杂查询）的使用：

# 中间打个逗号，意思是并且
stus = Student.objects.filter(Q(age=18),Q(gender=1))
# & 并且，对应and
stus = Student.objects.filter(Q(age=18) & Q(gender=1))
# | 或者，对应or
stus = Student.objects.filter(Q(age=18) | Q(gender=1))
#查询age不是18，或者gender=1的学生对象
stus = Student.objects.filter(~Q(age=18) | Q(gender=1))

# 查询语文成绩大于数学成绩的对象
stus = Student.objects.filter(yuwen__gt=F('shuxue'))
# 查询语文成绩大于数学成绩10分以上的
stus = Student.objects.filter(yuwen__gt=F('shuxue')+10)
